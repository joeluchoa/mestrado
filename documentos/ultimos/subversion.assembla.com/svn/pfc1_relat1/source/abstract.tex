%This project aims to study the Shortest Path Problem with Forbidden Paths addressing some variations of this problem and the relationship between them. Some solutions already proposed will be studied, implemented and compared to allow choice of best algorithm for each specific case of use. As primary target, this work aims to find the solution that best fits the problem of choosing the shortest path in a mesh of urban traffic.

Este projeto tem por objetivo estudar o Problema do Caminho Mínimo com Restrição de Sub-Caminhos, abordando algumas variações deste problema e a relação entre eles. Algumas soluções já propostas serão estudadas, implementadas e comparadas a fim de possibilitar a escolha do melhor algoritmo para cada caso de uso específico. Como alvo principal, este trabalho visa encontrar a solução que mais se adequa ao problema de escolha de menor caminho em uma malha de trânsito urbana.
