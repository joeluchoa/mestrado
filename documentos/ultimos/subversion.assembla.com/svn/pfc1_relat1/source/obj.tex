% ======================
% Objetivo do Projeto
% ======================

O objetivo deste projeto é estudar o Problema do Caminho Mínimo com Restrição de Subcaminhos (\emph{SPPFP -- Shortest Path Problem with Forbidden Paths}), e, principalmente, encontrar a solução que mais se adequa ao problema de escolha de menor caminho em uma malha de trânsito urbana. Este estudo abordará algumas variações do SPPFP (com diferentes configurações de restrições de subcaminhos) e a relação entre os mesmos. Além disso, serão estudadas propostas de algoritmos que solucionam tais problemas e implementadas algumas destas soluções, com o intuito de realizar comparações entre a eficiência dos algoritmos implementados. A interpretação dos  resultados obtidos nesses testes possibilitará a escolha do melhor algoritmo ou da melhor combinação de algoritmos para a solução de problemas reais, como, por exemplo, a escolha do menor caminho entre dois pontos numa malha de trânsito urbana. \cite{Szeider:2003}.
