\centerline{\bf Resumo}

\vspace{0.4cm}

\noindent
Esta dissertação trata de algoritmos para geração
de $k$-menores caminhos em grafos simétricos com custos não-negativos, 
bem como algumas implementações destes. 
O enfoque está no algoritmo desenvolvido por Naoki Katoh,
Toshihide Ibaraki e H. Mine, uma melhoria do algoritmo proposto por Yen, mas exclusiva para grafos simétricos.

\vspace{2.0cm}

\centerline{\bf Abstract} \vspace{0.4cm}

\noindent
This dissertation deals with $k$-shortest simple paths algorithms 
designed for non-negative costs, undirected graphs and 
some implementations of them. 
We focus on the algorithm developed by Naoki Katoh,
Toshihide Ibaraki e H. Mine, an enhanced version of Yen's algorithm, but valid only for nondirected graphs.
